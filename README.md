<summary>⚙️ How to enable the snake animation</summary>

The snake graph above is generated by a GitHub Action. To make it work, add this workflow at
`.github/workflows/snake.yml` in your profile repo (`darulqutni/darulqutni`):

```yaml
name: Generate Snake

on:
  schedule:
    - cron: "0 0 * * *"   # runs once a day
  workflow_dispatch:
  push:
    branches:
      - main

jobs:
  generate:
    runs-on: ubuntu-latest
    permissions:
      contents: write
    steps:
      - uses: Platane/snk@v3
        with:
          github_user_name: ${{ github.repository_owner }}
          outputs: |
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark

      - uses: crazy-max/ghaction-github-pages@v4
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```
